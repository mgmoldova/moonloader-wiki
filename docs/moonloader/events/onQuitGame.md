Выполняется в скрипте перед его нормальным завершением.

---

## Синтаксис

```lua
onExitScript(bool quitGame)
```

## Параметры

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `bool` | `quitGame` | будет иметь значение true, если скрипт был завершён в связи с выходом из игры |

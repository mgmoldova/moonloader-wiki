# main

Главная точка входа в скрипт. Вызывается ровно один раз в момент инициализации и запуска Lua-скрипта движком *MoonLoader*.

---

## Синтаксис

```lua
function main()
```

## Описание

Функция `main` — это фундамент любого некоррелируемого скрипта. В отличие от обычных обратных вызовов (событий сети или интерфейса), внутри `main` разрешено использовать бесконечные циклы и приостанавливать выполнение кода с помощью функции `wait()`.

!!! info "Архитектурная особенность"
    В действительности main является не просто стандартным событием, а сопрограммой (coroutine), выполняющейся в отдельном Lua-потоке. Именно это позволяет прерывать её выполнение на определённый промежуток времени (или количество кадров) и затем бесшовно продолжать работу с того же места без каких-либо негативных последствий для стабильности игры.

!!! danger "Критически важно: Бесконечные циклы и wait(0)"
    Если в функции main используется бесконечный цикл (while true do), в нём обязательно должна присутствовать хотя бы минимальная задержка — вызов wait(0).

Весь процесс игры, включая рендеринг кадров и скрипты MoonLoader, выполняется последовательно в одном основном потоке. Если вы запустите бесконечный цикл без `wait`, управление никогда не вернется к игровому движку, и **игра намертво зависнет**.

Вызов `wait(0)` приостанавливает выполнение скрипта ровно на один кадр, давая игре возможность обработать внутренние процессы. При этом задержка в `main` никак не влияет на обработку других асинхронных событий в этом же скрипте.
Пример использования

```lua
local script = require('moonloader')

function main()
    -- Предварительная загрузка ресурсов перед запуском основного цикла
    loadStuff()
    local key1_timesPressed = 0

    -- Бесконечный цикл — здесь реализуется основная логика работы скрипта
    while true do
        -- Обязательная нулевая задержка (пауза на 1 кадр) для предотвращения зависания
        wait(0)

        -- Проверяем, что персонаж игрока загружен и активен
        if isPlayerPlaying(playerHandle) then
            
            -- Проверяем нажатие клавиши "1" (0x31)
            if isKeyDown(0x31) then
                key1_timesPressed = key1_timesPressed + 1
                print("Key 1 pressed " .. key1_timesPressed .. " times.")
                
                -- Задержка 500 мс, чтобы одно нажатие не засчитывалось многократно за пару кадров
                wait(500)
            end

            -- Выполнение дополнительной фоновой логики
            doCoolStuff()
        end
    end
end
```

# isCursorActive

Проверяет, активен ли в данный момент игровой курсор (вызванный через `showCursor` или средствами SAMP).

---

## Синтаксис

```lua
bool result = isCursorActive()
```

## Параметры

Нет параметров

## Возвращаемые значения

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `bool` | `result` | описание |



!!! info "Связанные функции"
    * Чтобы принудительно показать или скрыть курсор, используйте функцию showCursor(bool state, bool toggleControls).

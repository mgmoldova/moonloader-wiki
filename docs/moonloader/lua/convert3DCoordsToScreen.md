# convert3DCoordsToScreen

Преобразует `convert3DCoordsToScreen`.

---

## Синтаксис

```lua
float wposX, float wposY = convert3DCoordsToScreen(float posX, float posY, float posZ)
```

## Параметры

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `float` | `posX` | описание |
| `float` | `posY` | описание |
| `float` | `posZ` | описание |

## Возвращаемые значения

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `float` | `wposX` | описание |
| `float` | `wposY` | описание |

# getPickupPointerHandle

Возвращает хэндл пикапа по указателю на экземляр класса CPickup.

---

## Синтаксис

```lua
int handle = getPickupPointerHandle(Pickup pickup)
```

## Параметры

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `Pickup` | `pickup` | хэндл пикапа |

## Возвращаемые значения

| Тип | Название | Описание |
| :--- | :--- | :--- |

| `int` | `handle` | указатель |

# getPickupPointer

Функция `getPickupPointer` из API MoonLoader.

---

## Синтаксис

```lua
int pointer = getPickupPointer(Pickup pickup)
```

## Параметры

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `Pickup` | `pickup` | указатель на пикап |

## Возвращаемые значения

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `int` | `pointer` | указатель на объект пикапа |

# script.get

Возвращает `script.get`.

---

## Синтаксис

```lua
LuaScript script = script.get(int scriptId)
```

## Параметры

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `int` | `scriptId` | описание |

## Возвращаемые значения

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `LuaScript` | `script` | описание |

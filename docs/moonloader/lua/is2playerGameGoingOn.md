# is2playerGameGoingOn

Проверяет, возвращает true, если функцию `is2playerGameGoingOn`.

---

## Синтаксис

```lua
bool result = is2playerGameGoingOn()
```

## Параметры

Нет параметров

## Возвращаемые значения

| Тип | Название | Описание |
| :--- | :--- | :--- |
| `bool` | `result` | описание |
